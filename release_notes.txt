## Breaking Changes

- Spell Book now requires the 3DS:ATLAS module, which provides shared theming, logging, localization, and update notices.
- Theming now runs through ATLAS: the spell book palette maps to ATLAS theme variables and defaults to the spellbook preset; the in-module logging-level and troubleshooter settings are gone.

## Bug Fixes

- SRD spell lists no longer vanish from the Spell List Manager when it is reopened without reloading the page.
- The unchecked spell prepare checkbox border is more visible on dark themes.

## Changes

- Bug reports run through the ATLAS troubleshooter, which attaches the enabled dnd5e compendium sources and every configured spell list for support.
- Bumped the bug-report template's version placeholders to Foundry 14.364 and dnd5e 6.0.0.
- Release and update notices are posted through ATLAS instead of an in-module chat card.
- Removed the bundled localization files; translations now come from the required 3DS:ATLAS module.
- Removed the redundant Game System field from the bug report template.
- Sidebar, filter checkbox, and property-toggle colors now follow the active ATLAS theme instead of fixed values.
- Standardized the GitHub issue templates on canonical type labels and added a Discord contact link to the issue chooser.
- Verified compatibility raised to Foundry 14.364.
