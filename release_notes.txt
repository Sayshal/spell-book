## New Features

- A class or subclass that gains spellcasting with no spell list assigned now sends a whispered chat card to the GM and the character's owners naming the affected classes, with a button that opens Class Rules for a GM and the Spell Book for a player.
- A class tab whose assigned list resolves empty because its sources are disabled now shows a "No spells available" notice instead of the generic no-list message.
- A list already assigned to a class stays in the Class Rules dropdown labeled "(source disabled)" when its sources are off.
- Party recommendations now flag missing healing, damage-type concentration, save concentration, short-range reliance, and costly material components.
- Spell lists whose spells all come from sources disabled in the dnd5e Compendium Browser are now hidden from the Class Rules dropdown and the Spell List Manager sidebar; custom, merged, and actor-owned lists stay visible.
- The Party Spell Synergy Analysis shows a Saving Throws spread beside Damage Types, with the left column ordered Damage Types, Saving Throws, Spell Levels, School.
- The Spell List Manager footer shows a count of lists hidden by disabled sources while browsing lists.

## Bug Fixes

- A single malformed compendium pack no longer blanks the Class Rules spell-list dropdown; the bad pack is skipped with a warning and the remaining spell lists still appear.
- Fixed the Damage Types and Saving Throws spreads reading blank under dnd5e 5.x; both now populate from spell activities.
- Saving Class Rules no longer wipes a class's assigned list when that list's sources are disabled.
- The GM whisper for spell changes is now delivered; it fires once per save when "Notify GM on Spell Changes" is enabled and a player adds or removes any prepared cantrip or spell.

## Changes

- Restyled the spell-change chat card with an arcane themed layout and a single alphabetical Removed/Added table; it lists removed and added spells as content links with a level column and shows an over-limit warning line when a class exceeds its cantrip or prepared spell limit.
