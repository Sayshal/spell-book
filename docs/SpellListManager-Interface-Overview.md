# SpellListManager Interface Overview

GM-only tool for creating, modifying, and managing spell lists.

---

## Accessing Spell List Manager

Only GM (or GM Assistant) users can access the **Spell List Manager**. Open it from the **Compendium** tab — click the **Spell List Manager** button at the bottom of the sidebar.

---

## Interface Layout

The Spell List Manager is divided into three panels:

- **Left Sidebar** — Spell list navigation
- **Center Panel** — Spell list contents and editing
- **Right Panel** — Spell selection and filtering tools

---

## Spell List Navigation (Left Sidebar)

At the top-left, next to **Available Spell Lists**, are two action buttons:

- **Merge** — Combine two existing spell lists into a new custom list
- **Create** — Create a new custom spell list from scratch

Below these buttons, lists are organized into three sections:

### Player Spell Book

Lists tied to wizard characters or characters with **Force Wizard Mode** enabled. These represent spells the character has learned and currently has access to.

### Custom Spell Lists

Lists you have created manually or generated by merging. Fully editable. Typically used to define class, subclass, or homebrew spell availability.

### Spell Lists

Default lists provided by the game system and enabled modules. Left-click a list to load it into the center panel.

---

## Spell Management Panel (Center Panel)

Displays the contents of the selected spell list. Click **Edit** at the top to enable editing. While in edit mode, you can add, remove, or reorganize spells. Changes from the right panel are reflected in real time.

---

## Spell Selection and Tools (Right Panel)

Browse, filter, and add spells to the selected list:

- Search and filter spells from configured compendiums
- Click the **+** icon next to a spell to add it individually

### Selection Mode

Toggle at the bottom-right corner of the window. When enabled:

- **+** icons are replaced with checkboxes
- Select multiple spells at once
- Add all selected spells in a single action

Useful when building or updating large spell lists.

---

## Dialogs

### Spell Comparison

Each spell entry includes a blue clipboard icon. Click it on two or more spells to open the Spell Comparison dialog, showing key info side by side (damage type, casting level, spell school).

### Spell Details Customization

Click **Configure Display** in the bottom-right corner to open the customization dialog:

- Customize which spell details are shown
- Hide specific fields to reduce clutter
- Adjust display options for both the Spell Book and Spell List Manager

---

## Common Workflows

### Creating Custom Spell Lists

1. Open the Spell List Manager from the **Compendium** tab
2. Click **Create** next to **Available Spell Lists**
3. Provide a name for the new list
4. Select it from the left sidebar
5. Click **Edit** in the center panel
6. Use the right panel to browse and filter spells
7. Add spells individually (**+** icon) or enable **Selection Mode** for bulk adds
8. Review the list in the center panel
9. Exit edit mode when finished

Custom lists appear under **Custom Spell Lists** and can be assigned to characters through their Spell Book settings.

### Modifying Existing Spell Lists

1. Select a spell list from the left sidebar
2. Review contents in the center panel
3. Click **Edit** to enable modifications
4. Add or remove spells as needed
5. Exit edit mode to save changes

> [!NOTE]
> System-provided lists are read-only. When you edit one, Spell Book automatically creates a custom copy so your changes are preserved across module updates.

### Managing Player Access

1. Enable **GM Setup Mode** while making bulk changes
2. Create or adjust spell lists in the Spell List Manager
3. Open a player character's sheet
4. Access their Spell Book settings (wand icon)
5. Assign one or more spell lists
6. Verify the correct spells are available
7. Disable **GM Setup Mode** when done

Players can only learn, prepare, or interact with spells on their assigned lists.
