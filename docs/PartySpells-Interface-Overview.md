# PartySpells Interface Overview

A specialized window for viewing and coordinating spells across all party members. Helps optimize spellcasting roles and reduce redundant preparations in multi-caster parties.

---

## Group Actor Sheet Integration

Open the Party Spell Interface using the book icon on the **Group Actor Sheet** tab to view all party members' prepared spells.

- Spells are displayed per level with notes indicating who has each spell prepared
- Real-time updates as players change spell preparation
- Synchronization of spells across party members
- Focus selection to communicate preparation intentions

---

## Focus Settings

The Focus Settings dialog allows players and GMs to configure spellcasting focus distribution across the party.

- Assign roles or priorities to spellcasters (healer, damage dealer, buffer, etc.)
- Players can set their own focus, or the GM can assign one
- GMs can add additional custom focus options

---

## Party Synergy Analysis

Automated analysis of party spell composition:

- Lists duplicate prepared spells to suggest changes
- Shows which damage types are represented and which are underrepresented
- Highlights gaps in healing, spell schools, and preparation by spell level
