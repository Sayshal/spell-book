# DM Quick Start

## Getting Started as a GM

### Spell List Manager Overview

Your spell list management tool can be accessed by the **Spell List Manager** button at the bottom of your compendium tab.
This tool is where you will create, modify, and delete spell lists.

- Full Reference of All information can be found here: [Spell List Manager - Interface Overview](SpellListManager-Interface-Overview)

### Spellbook Settings Dialog

In the Settings Menu, Spell Book adds serveral DM facing settings. You do not need to configure everything immediately.

Core Settings:
- Spellcasting Rule Set used (Defaults to Legacy)
- Enforcement Behavior (Defaults Notify GM)
- Setup Mode (Recommended to have off during sessions)
 >GM Setup Mode preloads all spells and compendiums to reduce loading time while setting things up. Always disable it once setup is complete.

Spell Management Options:
- Consume Scrolls when Learning (Default enabled)
- Deduct Spell Learning Costs (Default disabled)
- Enable Spell Usage Analytics (Default Enabled)

Interface Options:
- Spell Comparison Maximum (Default 3)
- Cantrip Scale Values (Default: "Cantrips-known, cantrips")

Advanced Settings:
- Suppress Migration Warnings (Default disabled)
- Cauldron of Plentiful Resources Compatibility (Default disabled)
- Auto-Delete Unprepared Spells (Default disabled)

For a full list of the settings and what they do visit the refrence below.
- Reference: [Installation and Settings](Installation-and-Settings)

Common GM workflow patterns:
- Adjusting Settings and enabling **GM Setup Mode** 
- Creating New Spell Lists for the Classes/Subclasses being used in the game using Spell List Manager
- Using Player spell book settings (accessible on their character sheets) to adjust what spell lists they are using
- Turn off **GM Setup Mode** when finished

## Initial Setup Recommendations

For new campaigns, here are the recommended configurations:
- Configure Compendiums: in this menu select only the compendiums you are using the spells from to reduce loading lag. By default it should be one of the SRD's and any Spell Adding Modules you use.
- Enable Spell Usage Tracking: ON
- Auto-Delete Unprepared Spells: OFF
- Consume Scrolls When Learning: ON
- Deduct Spell Learning Costs: OFF
- Disable Long Rest Swap Prompt: OFF
- Suppress Migration Warnings: OFF
- Cauldron of Plentiful Resources Compatability: ON if you are using CPR, IF NOT USING DONT TURN IT ON.
